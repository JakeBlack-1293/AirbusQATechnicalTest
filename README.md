# AirbusQATechnicalTest



In this repository is the Automation Test for the following Techincal Test. 

Technical Interview Task Using Java Selenium and Cucumber/Gherkin, please implement the following test:
1. Open a browser, and navigate to the Airbus Careers Website.
2. Follow the link to the Workday Jobs List.
3. Using the filters, locate the role you have applied for.
4. Click Apply, on that role, and detect that the pop up is present on the screen.

The automation test uses Java/Selenium/Cucumber as well as implementing the Page Object Model standard. All tests passed as can be seen by the generated report in the repository or via the screenshot below.
